AirIQ Module

Air Quality Monitoring S360-AIR-C / S360-AIR-W

Comprehensive air quality monitoring with multiple sensors for health-conscious environments.

Sensor Measurements Range I2C Address
SPS30 PM1.0, PM2.5, PM4.0, PM10 0-1000 µg/m³ 0x69
SGP41 VOC Index, NOx Index 0-500 index 0x59
SCD41 CO2, Temperature, Humidity 400-5000 ppm 0x62
BMP390 Barometric Pressure 300-1250 hPa 0x77

Comfort Module

Environmental Comfort S360-CMFT-C / S360-CMFT-W

Basic environmental comfort sensing for temperature, humidity, and ambient light.

Sensor Measurements Range / Accuracy I2C Address
SHT40 Temperature, Humidity -40°C to +125°C, ±0.2°C / 0-100% RH, ±1.8% 0x44
LTR-303 Ambient Light 0.01-64k lux 0x29

Presence Module

mmWave Radar Presence S360-PRES-C / S360-PRES-W

Radar-based presence and occupancy detection for automation.

Sensor Features Range Interface
HLK-LD2450 Multi-target tracking (3), still/moving, zone-based Up to 6m UART 256000
DFRobot C4001 24GHz FMCW, speed measurement 16m presence, 25m motion UART
HLK-LD2412 Single-zone, gate thresholds, better still detection Up to 6m UART 115200

Bathroom Module

Bathroom Specialty S360-BATH-B / S360-BATH-P

Optimized for bathroom environments. Ceiling mount only. Replaces AirIQ (cannot be used together).

Base Variant (S360-BATH-B)

Sensor Measurements Purpose
SHT4x Temperature, Humidity Shower detection, humidity spikes
BMP390 Barometric Pressure Environmental baseline
SGP41 VOC/NOx Index Odor detection

Pro Variant (S360-BATH-P) — Additional Sensors

Sensor Measurements Purpose
MLX90614 IR Surface Temperature Condensation risk detection
SPS30 PM1.0, PM2.5, PM10 Particulate monitoring

Fan Control Module

HVAC Fan Control S360-GP8403 / S360-PWM

Fan speed control for ventilation automation.

Module Output Resolution Use Case
GP8403 DAC 0-10V or 0-5V analog 12-bit (0.01V) Commercial HVAC, EC motors, VFDs
PWM 25kHz PWM signal 8-bit Standard fans, 4-pin PC fans
