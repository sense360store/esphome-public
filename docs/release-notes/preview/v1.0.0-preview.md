# Sense360 shared preview firmware v1.0.0-preview

`v1.0.0-preview` is the shared GitHub prerelease for Sense360 preview firmware
artifacts. This source covers the room preview artifacts, the LED preview
artifact, and the FanRelay / FanPWM / FanDAC manual-preview artifacts now
co-hosted on that same release.

Preview/manual-preview firmware only: not stable, not recommended, not default,
not Simple install, and not buyable as public shop product. No hardware proof.
No bench proof. No compliance proof. No commercial-availability proof.
Fan-control builds require installer/developer judgement. TRIAC excluded.

Presence on this shared release does not imply WebFlash import for fan
artifacts, Simple-install exposure, recommendation, default selection, stable
status, hardware proof, compliance proof, or commercial availability.

## Changelog

- Reconciles the shared `v1.0.0-preview` preview release body so it covers all
  co-hosted preview artifacts: four room preview artifacts, including the LED
  preview artifact, plus three fan manual-preview artifacts.
- Room preview artifacts:
  - `Ceiling-POE-AirIQ-RoomIQ`
  - `Ceiling-POE-RoomIQ`
  - `Ceiling-POE-RoomIQ-LED`
  - `Ceiling-POE-VentIQ-RoomIQ-LED`
- Fan manual-preview artifacts:
  - `Ceiling-POE-VentIQ-FanRelay-RoomIQ`
  - `Ceiling-POE-FanPWM`
  - `Ceiling-POE-FanDAC`
- Excluded:
  - `Ceiling-POE-VentIQ-FanTRIAC-RoomIQ`
- The fan manual-preview artifacts are co-hosted on the shared GitHub release
  only; they are not WebFlash imports, not Simple install, not recommended, not
  default, not stable, and not buyable as public shop product.

## Known Issues

- Preview/manual-preview firmware only: not stable, not recommended, not
  default, not Simple install, and not buyable as public shop product.
- No hardware proof; no bench proof; no compliance proof; no
  commercial-availability proof.
- Fan-control builds require installer/developer judgement. Installer wiring,
  load suitability, recovery planning, and target-specific fan safety review
  remain outside the proof supplied by this preview release.
- Fan manual-preview artifacts do not imply WebFlash import for fan artifacts.
  No fan row is added to `config/webflash-builds.json`, no WebFlash repo change
  is made, and no Simple-install exposure is created.
- TRIAC excluded: `Ceiling-POE-VentIQ-FanTRIAC-RoomIQ` remains out of the shared
  release because `HW-005` still blocks buildability. No TRIAC preview artifact
  is included.

## Features

- Shared preview-release source for the four room preview artifacts:
  `Ceiling-POE-AirIQ-RoomIQ`, `Ceiling-POE-RoomIQ`,
  `Ceiling-POE-RoomIQ-LED`, and `Ceiling-POE-VentIQ-RoomIQ-LED`.
- Shared preview-release source for the three fan manual-preview artifacts:
  `Ceiling-POE-VentIQ-FanRelay-RoomIQ`, `Ceiling-POE-FanPWM`, and
  `Ceiling-POE-FanDAC`.
- Room preview coverage remains the WebFlash-preview side of the shared release.
  Fan manual-preview coverage remains manual-preview only and separately gated
  from WebFlash import decisions.
- LED preview coverage remains preview-only and does not change any Simple
  install, recommendation, default, stable, shop, or proof posture.

## Hardware Requirements

- Room preview artifacts require the matching Sense360 Core / PoE / room-module
  combination for each listed room configuration; LED-bearing room preview
  artifacts also require the Sense360 LED hardware.
- `Ceiling-POE-VentIQ-FanRelay-RoomIQ` requires the matching VentIQ, RoomIQ, and
  FanRelay hardware and installer/developer judgement for the switched fan load.
- `Ceiling-POE-FanPWM` requires the matching FanPWM hardware and
  installer/developer judgement for the PWM fan load.
- `Ceiling-POE-FanDAC` requires the matching FanDAC hardware and
  installer/developer judgement for the analog fan-control load.
- `Ceiling-POE-VentIQ-FanTRIAC-RoomIQ` is excluded and has no hardware
  requirement in this shared release because no TRIAC artifact is included.
