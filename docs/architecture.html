System Architecture

How Sense360 works. A modular smart home sensor platform built around an ESP32-S3 core with stackable sensor modules.

CORE + POWER + LED (optional) + MODULES = Your Sense360

Choose a core board, pick a power source, add an LED ring, and stack any combination of sensor modules.

Hardware Layers

Core Board Required

The ESP32-S3 base that connects all modules and provides WiFi connectivity.

  • Sense360 Core Ceiling S360-CORE-C
  • Sense360 Core Wall S360-CORE-W
  • Sense360 Mini S360-MINI

Power Module Required

How the system is powered. Choose based on your installation needs.

  • USB-C (built-in) —
  • Power over Ethernet S360-POE
  • 240V AC Mains S360-PWR

LED Ring Optional

Visual feedback for air quality, presence, and system status.

  • LED Ring Ceiling S360-LED-C
  • LED Ring Wall S360-LED-W

Sensor Modules Optional

Stack any combination of environmental sensors.

  • AirIQ (Air Quality) S360-AIR-*
  • Comfort (Temp/Humidity) S360-CMFT-*
  • Presence (Radar) S360-PRES-*
  • Bathroom (Ceiling only) S360-BATH-*
  • Fan Control S360-GP8403/PWM

Sense360 Mini

The Mini is a compact all-in-one board with sensors integrated directly on the PCB — no expansion modules needed for basic environmental monitoring.

Communication Interfaces

I2C Bus

Used by most sensors for data communication.

  • SDA Pin (Mini) GPIO48
  • SCL Pin (Mini) GPIO45
  • Speed 400kHz

UART

Used by presence radar sensors.

  • TX Pin (Mini) GPIO43
  • RX Pin (Mini) GPIO44
  • LD2412 Baud 115200
  • LD2450 Baud 256000

LED Data

Addressable LED control for status indicators.

  • Data Pin (Mini) GPIO8
  • Protocol WS2812B
  • Color Order GRB

WiFi / API

Home Assistant integration via ESPHome.

  • Protocol ESPHome Native API
  • Encryption AES-256
  • Discovery mDNS
