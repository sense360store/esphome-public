## Status

`Sense360 PoE PSU` board, revision **R4** (`S360-410-R4`). It
exists to close the documentation gap recorded at

## Known assumptions

Stated as assumptions, not as facts. None of these is sufficient
to mark `S360-410` `verified`, to promote the preserved
caveat away, or to resolve HW-002 OQ#6.

- **The module accepts 802.3af / 802.3at PoE and produces 5 V
  DC.** The catalog description records `PoE to 5V.`. The
  package header records `IEEE 802.3af (PoE) or 802.3at (PoE+)`
  with `36-57V DC` input and `5V DC, 2A (10W) or 3.3V DC`
  output. Neither is schematic-evidenced for `S360-410-R4`.
- **The module is off-board relative to Core.** The Core
  schematic captures only a 2-pin `PoE_ACDC` inlet at `J2`; the
  PoE magnetics, RJ45, and PD controller IC live on the
  off-board module.
- **`PoE_ACDC` enters the Core at `J2` pin 1 and `GND` at `J2`
  pin 2.** Captured in
  [`s360-100-r4-core.md` §J2](s360-100-r4-core.md#j2--poe_acdc-inlet-2-pin).
- **`PoE_ACDC` joins the Core `+5V` rail through `Q2 Si2319CDS`
  + `D12` ideal-diode-OR with `VBUS`.** Captured in
  [`s360-100-r4-core.md` Power inputs and rails](s360-100-r4-core.md#power-inputs-and-rails).
- **PoE is SELV (safety extra-low voltage).** PoE is not
  mains-side; the 36–57 V DC PoE input is galvanically isolated
  from mains at the PSE (switch / injector) side. **For this
  reason `S360-410` is not in scope for COMPLIANCE-001** (the
  mains-voltage UK / EU compliance tracker scopes `S360-320`
  FanTRIAC and `S360-400` 240 V PSU only).
- **The off-board harness physically delivers `PoE_ACDC` and
  `GND` to Core `J2`.** HW-002 OQ#6 asks which physical
  connector / cable / pigtail the module uses to mate with `J2`.
  The harness pin order, conductor count, length, connector key
  / polarity at both ends are unknown without the module
  schematic and bench observation.

These assumptions are not invented rail / PoE-controller maps.
Anything beyond the list above — PD controller IC part identity,
802.3af signature and classification circuit, transformer /
magnetics construction, RJ45 indicator LEDs, on-board
diagnostics, exact regulator topology after the PoE bridge,
output capacitor stack, output protection (OVP / OCP / SCP),
in-rush, harness identity — is **unknown** and must come from
the module-side schematic plus bench evidence.
