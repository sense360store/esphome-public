# Compare products

!!! note "Comparison matrix coming"
    A full four-product comparison matrix (sensing capabilities, air
    quality metrics, outputs, and release channel) is generated from the
    product catalog in the next docs update (PRODUCT-GUIDES-001 G2), so it
    can never drift from what the firmware ships.

Until then, in one line each:

- **[Room sensing (RoomIQ)](ceiling-poe-roomiq.md)** — presence +
  comfort sensing only; the smallest configuration.
- **[Air quality + room sensing (AirIQ + RoomIQ)](ceiling-poe-airiq-roomiq.md)**
  — RoomIQ plus the AirIQ air-quality module.
- **[Bathroom (VentIQ + RoomIQ)](ceiling-poe-ventiq-roomiq.md)** — RoomIQ
  plus bathroom air-quality sensing with shower detection and mould-risk
  warnings.
- **[Bathroom + LED ring (VentIQ + RoomIQ + LED)](ceiling-poe-ventiq-roomiq-led.md)**
  — the bathroom product plus a 12-LED indicator ring (preview channel).

All four are PoE-powered ceiling products built on the Sense360 Core hub.
